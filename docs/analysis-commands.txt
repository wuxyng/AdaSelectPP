```bash
cd <repo-root>
```

```bash
python scripts/analyze_phase15_trace.py \
  --glob "runs_phase15_retainswap_firstpass/*/log/*.trace.csv" \
  --outdir analysis_phase15_trace
```

```bash
python scripts/analyze_phase15_trace.py \
  --glob "runs_phase15_retainswap_firstpass/*/log/*.trace.csv" \

  --outdir analysis_phase15_trace_curr_keep_retain_swap
```

筛 workload：

```bash
python scripts/analyze_phase15_trace.py \
  --glob "runs_phase15_retainswap_firstpass/*/log/*.trace.csv" \
  --bench tpchs \
  --wtype shifting \
  --outdir analysis_phase15_trace_tpchs_shifting
```

包含 rejected visible/evaluated rows：

```bash
python scripts/analyze_phase15_trace.py \
  --glob "runs_phase15_retainswap_firstpass/*/log/*.trace.csv" \

  --outdir analysis_phase15_trace
```

```bash
python scripts/analyze_phase15_trace.py \
  --glob "runs_phase15_retainswap_firstpass/*/log/*.trace.csv" \
```


```text
swap_blocked_partner
swap_blocked_gain
swap_blocked_delta
swap_blocked_old_score
swap_blocked_new_score
```
